# Consulta Diaria BOCM · v7

Aplicación desplegable en Vercel para:

- consultar un boletín por fecha y filtrar contenidos técnicos;
- excluir de forma estructural el apartado B) Autoridades y Personal;
- buscar palabras y nombres de proyectos en el histórico oficial del BOCM desde febrero de 2010;
- filtrar opcionalmente la búsqueda histórica por fechas y municipio;
- abrir únicamente la versión HTML de los anuncios.

## Actualización

Sustituir en GitHub los archivos incluidos en `bocm-cambios-v7.zip`. Vercel desplegará automáticamente el commit.
